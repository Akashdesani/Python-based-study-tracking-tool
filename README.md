# AI Study Tracker & Productivity Analyzer 📚⏱️

A powerful **Python-based study tracking tool** that helps students monitor their study sessions, track subject-wise time, and analyze productivity levels.

This project is designed to improve **study discipline and productivity** by recording daily study sessions and generating useful reports.

---

## 🚀 Features

* ⏱ **Study Timer** – Track real-time study sessions
* 📚 **Subject-wise Tracking** – Record time spent on different subjects
* 📊 **Daily Study Reports** – View study data for each day
* 🔥 **Productivity Score** – Calculate total study hours and productivity level
* 💾 **Automatic Data Storage** – Saves study data using JSON files
* 🧠 **Simple AI-style Analysis** – Evaluates productivity level automatically

---

## 🛠️ Technologies Used

* JSON (Data Storage)

## ▶ How It Works

1. Start a study session
2. Enter the subject name
3. Press **Enter** to start the timer
4. Press **Enter** again to stop the session
5. The system automatically saves your study time
6. Generate reports and check productivity score
